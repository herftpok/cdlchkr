# CDL Checker

Professional driver background screening service for commercial motor carriers.

## Features

- **Driver Background Checks**: MVR, PSP, CDLIS, Drug & Alcohol Testing
- **Package System**: Purchase check packages with volume discounts
- **Hybrid Payments**: Use packages or pay with card
- **Profile Management**: User settings, referral program, API access
- **Check History**: View and download all previous checks

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 3
- Lucide Icons

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
